# Paper Alert 使用教程

自动从学术期刊、NBER、WFA/AFA 会议中抓取论文，按关键词筛选后推送到微信或邮件。

---

## 一、安装

```bash
git clone https://github.com/Antongnidas/paper-alert.git
cd paper-alert
pip install -r requirements.txt
cp config.example.yaml config.yaml
```

---

## 二、配置（编辑 config.yaml）

### 1. 设置关键词

```yaml
keywords:
  include:
    - healthcare      # 至少命中一个才保留
    - mutual fund
    - insurance
  exclude:
    - editorial       # 命中就排除
    - corrigendum
```

### 2. 设置时间窗口

```yaml
lookback_days: 14   # 抓取过去14天内发表的论文
```

### 3. 选择期刊/来源

```yaml
selection:
  groups:
    - top_finance     # JF、JFE、RFS、JFQA
    - top_om          # Management Science、OR、MSOM
    - working_papers  # NBER Working Papers
    - conferences     # WFA、AFA 会议论文
```

可用的 group：

| group | 包含来源 |
|-------|----------|
| top_finance | JF, JFE, RFS, JFQA |
| top_om | MNSC, OR, MSOM |
| top_is | ISR |
| top_marketing | Marketing Science |
| top_management | Organization Science |
| top_accounting | JAE, JAR |
| om_extended | POM, JOM |
| econ_core | Econometrica, JPE, REStud（AER/QJE 无公开 RSS 暂不支持） |
| working_papers | NBER |
| conferences | WFA, AFA |

也可以按期刊名精确选择：

```yaml
selection:
  source_names:
    - "Journal of Finance"
    - "NBER Working Papers"
```

### 4. 设置推送（二选一或都开）

**微信推送（推荐，最简单）：**
1. 访问 [sct.ftqq.com](https://sct.ftqq.com/)，微信扫码登录
2. 复制你的 SendKey
3. 填入 config.yaml：

```yaml
notify:
  wechat:
    enabled: true
    sckey: "你的SendKey"
```

**邮件推送（Gmail）：**
1. 开启 Google 两步验证
2. 访问 [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords) 生成 App Password（16位字母，**不是**登录密码）
3. 填入 config.yaml：

```yaml
notify:
  email:
    enabled: true
    smtp_host: smtp.gmail.com
    smtp_port: 465
    sender: "你的邮箱@gmail.com"
    password: "16位AppPassword"
    recipients:
      - "收件人@gmail.com"   # 可以填自己，也可以填别人
```

> **校园网/公司网用户：** SMTP 端口通常被封锁。程序会自动读取 git 的代理配置并走代理发邮件，只需确保代理正在运行，无需额外设置。
> 如果还没配置 git 代理，运行：
> ```bash
> git config --global http.proxy http://127.0.0.1:你的代理端口
> ```

## 三、运行

```bash
python main.py
```

结果保存在 `outputs/` 目录：
- `papers.md` — Markdown 格式，可直接阅读
- `papers.csv` — 表格格式
- `papers.json` — 完整数据

## 四、会议论文（WFA / AFA）

### 自动查找最新议程

```bash
python main.py --find-agenda wfa
python main.py --find-agenda afa
```

程序会自动爬取官网找到最新年份的议程 PDF，询问是否写入 `sources.yaml`，输入 `y` 确认即可。

> AFA 官网有时不直接挂 PDF，可手动下载议程 PDF 后填入 `sources.yaml` 的 `url` 字段。

### 开启/关闭某个会议

打开 `sources.yaml`，找到对应条目，修改 `enabled`：

```yaml
- name: "WFA 2025"
  enabled: true    # true=开启，false=关闭
```

### 下载匹配论文的 PDF

在 `sources.yaml` 中把 `download_pdf` 改为 `true`：

```yaml
- name: "WFA 2025"
  download_pdf: true
  pdf_output_dir: "outputs/pdfs/WFA2025"
```

运行后，关键词命中的论文 PDF 会自动下载到指定目录。

## 五、常见问题

**Q：运行后没有匹配到论文？**
- 检查 `lookback_days` 是否太小
- 检查关键词拼写（目前只匹配标题，全词匹配）
- 可以在 `matching` 里改为 `title_and_abstract` 扩大范围

**Q：微信收不到推送？**
- 检查 `sckey` 是否正确
- 确认 `enabled: true`

**Q：邮件发送失败？**
- 确认使用的是 App Password（16位），不是 Gmail 登录密码
- 校园网/公司网封锁了 SMTP 端口时，程序会自动走 git 代理。确认代理正在运行，并已配置：
  ```bash
  git config --global http.proxy http://127.0.0.1:你的代理端口
  ```

**Q：NBER 论文抓不到？**
- NBER 需要代理访问，确认代理正常运行

## 六、分享给别人使用

只需要修改 `config.yaml`，其他文件不用动：

| 配置项 | 说明 |
|--------|------|
| `keywords.include` | 换成自己的研究关键词 |
| `keywords.exclude` | 保持默认或自定义 |
| `lookback_days` | 时间窗口，默认14天 |
| `selection.groups` | 选择要监控的期刊/会议组 |
| `notify.wechat.sckey` | 自己的 Server酱 SendKey |
| `notify.email` | 自己的 Gmail 和 App Password |

`sources.yaml` 和所有代码文件**不需要修改**。
